🚀 БЫСТРЫЙ СТАРТ ДЛЯ WINDOWS

═══════════════════════════════════════════════════════════

📦 ЧТО УСТАНОВИТЬ (только первый раз):

1. Node.js - скачать с https://nodejs.org/
2. PostgreSQL - скачать с https://www.postgresql.org/download/windows/

═══════════════════════════════════════════════════════════

⚙️ УСТАНОВКА FUSE:

1. Запустите: INSTALL_WINDOWS.bat
2. Подождите 2-3 минуты
3. Готово!

═══════════════════════════════════════════════════════════

▶️ ЗАПУСК FUSE:

1. Запустите: START_FUSE.bat
2. Браузер откроется автоматически!
3. НЕ ЗАКРЫВАЙТЕ черные окна!

═══════════════════════════════════════════════════════════

🎮 ИСПОЛЬЗОВАНИЕ:

При первом запуске:
- Адрес: localhost
- Порт: 3000
- Нажмите "Подключиться"
- Зарегистрируйтесь

Что умеет Fuse:
✅ Текстовый чат
✅ Голосовые звонки
✅ Видео звонки
✅ Отправка файлов (фото, видео, музыка)
✅ GIF аватарки
✅ Групповые звонки

═══════════════════════════════════════════════════════════

👥 ИГРАТЬ С ДРУЗЬЯМИ:

Вариант 1: Одна сеть Wi-Fi
- Узнайте свой IP: Win+R → cmd → ipconfig
- Друзья заходят: http://ваш-ip:5173

Вариант 2: Hamachi
- Скачайте Hamachi с vpn.net
- Создайте сеть
- Друзья заходят: http://hamachi-ip:5173

Вариант 3: Radmin VPN (бесплатно!)
- Скачайте с radmin-vpn.com
- Создайте сеть
- Друзья заходят: http://radmin-ip:5173

═══════════════════════════════════════════════════════════

🛑 ОСТАНОВКА:

Запустите: STOP_FUSE.bat
Или закройте черные окна
